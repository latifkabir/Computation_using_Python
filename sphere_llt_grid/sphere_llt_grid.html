SPHERE_LLT_GRID
Latitude/Longitude Triangle Grids on a Sphere

SPHERE_LLT_GRID is a Python library which uses longitudes and latitudes to create grids of points, lines, and triangles over the surface of a sphere in 3D.

Languages:

SPHERE_LLT_GRID is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.
